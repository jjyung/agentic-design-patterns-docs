# Agentic 設計模式 - 完整參考

本目錄包含 21 個整合的 agentic 設計模式，每個模式結合了詳細討論與視覺化流程圖。

## 每個模式的內容

每個模式檔案包含：

- **何時使用**：特定情境和使用案例
- **視覺化流程**：展示流程的互動式 Mermaid 圖表
- **適用位置**：應用領域和情境
- **優點**：優勢和好處
- **缺點**：限制和挑戰
- **實際案例**：實用的實作情境
- **原始檔案**：來源文件參考

## 如何使用此參考

1. **依類別瀏覽**：從符合您需求的類別開始
2. **檢視視覺化流程**：每個模式都包含一個嵌入的 Mermaid 圖表
3. **檢查範例**：實際情境有助於理解實際應用
4. **審查權衡**：優缺點幫助您做出明智決策
5. **存取來源**：原始檔案可供詳細研究
